# Slides

This folder stores slide-related planning and presentation materials for VCE English. It is intended to support the creation of classroom slides, revision decks, SAC briefings, text-specific lessons, and workshop presentations.

## Purpose

The purpose of this folder is to separate visual teaching materials from lesson planning notes. This makes it easier to build, locate, and reuse slide content for classroom teaching.

This folder can be used to store:
- Slide outlines.
- Lesson deck plans.
- Revision slide structures.
- Exam briefing presentations.
- Workshop slide plans.
- Speaker notes.
- Prompt-based slide generation files.

## Recommended slide outline format

A slide outline in this repository should usually include:

1. Slide title
2. Lesson or workshop purpose
3. Key teaching points
4. Suggested visual or example
5. Student task or discussion prompt
6. Optional speaker note

## Slide design principles

When storing slide materials, aim to keep them:
- Minimal in wording.
- Clear in sequence.
- Focused on one main idea per slide.
- Easy to convert into Canva, PowerPoint, or Claude-generated slide decks.
- Built around explanation, modelling, and student practice.
